### 管程

- Java 采用的是管程技术，synchronized 关键字及 wait()、notify()、notifyAll() 这三个方法都是管程的组成部分

- 管程和信号量是等价的，所谓等价指的是用管程能够实现信号量，也能用信号量实现管程

- 管理共享变量以及对共享变量的操作过程，让他们支持并发。翻译为 Java 领域的语言，就是管理类的成员变量和成员方法，让这个类是线程安全的
